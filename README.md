# Medical Appointments Management System

This is a team project developed as part of our university coursework.  
The web application allows patients to book medical appointments and doctors to manage their schedules efficiently.

## Features
- Patient registration and login
- Booking and editing medical appointments
- Doctors’ schedule management
- Database storage for users, doctors and appointment data
- User-friendly web interface

## Technologies Used
- **Java Servlets** (Backend logic and request handling)
- **JSP** (User interface)
- **MySQL / SQL** (Database storage)
- **Maven** (Dependency management and project build)
- **Tomcat** (Application deployment)

## How to Run the Project
1. Import the project into **Eclipse / IntelliJ / VS Code** (with Maven support)
2. Run `create_insert.sql` in your MySQL database
3. Configure database connection credentials (if needed)
4. Deploy the project on **Apache Tomcat**
5. Open the application in your browser
